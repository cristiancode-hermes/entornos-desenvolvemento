# 📚 Contornos de Desenvolvemento — App de Estudo

App web interactiva para preparar o examen de **Entornos de Desenvolvemento** do martes 16 de xuño.

## 📋 Temario

| Tema | Contido |
|------|---------|
| **Tema 2** | Instalación e uso de Contornos de Desenvolvemento (IDEs, NetBeans, Eclipse, VS Code, IntelliJ) |
| **Tema 3** | Deseño e realización de probas (Testing, depuración, JUnit, JaCoCo) |
| **Tema 4** | Optimización e documentación (Refactorización, patróns de deseño, Git, Javadoc) |
| **Tema 5** | Elaboración de Diagramas de Clases (POO, UML, diagramas de clases) |
| **Tema 6** | Elaboración de Diagramas de Comportamento (Casos de uso, secuencia, estados, actividades) |

## 🎮 Funcionalidades

- **📖 Resumos** — Explicación detallada de cada tema con exemplos e táboas
- **🧪 Preguntas tipo test** — 10 preguntas por tema (50 en total) con feedback inmediato
- **🏆 Sistema de progreso** — XP, niveis, estatísticas e raia de acertos
- **💾 Gardado automático** — O progreso mantense no navegador (localStorage)
- **🌙 Tema claro/escuro** — Cambia entre modo claro e escuro

## 🚀 Como usar

Abre o ficheiro `index.html` en calquera navegador moderno. Non precisa instalación nin servidor.

Ou en liña: https://entornos-dev.netlify.app (se está deployado)

## 💡 Consellos de estudo

1. Lee o **resumo** de cada tema primeiro
2. Fai as **preguntas** para comprobar o que sabes
3. Repasa os conceptos que falles
4. Repite ata sacar +70% en todos os temas

Boa sorte co examen! 🎯
